Pizza_sales_analysis (Power BI & SQL Project)

This is an Power BI-based sales analysis project created to explore 2015 sales performance. Using tools like Power Query, SQL, DAX Functions, the project provides business insights through a dynamic sales dashboard.

Tools & Features Used

Power BI
Power Query for data cleaning and transformation
SQL Queries GROUP BY, ORDER BY, SUM, AVG, LIMIT, Percentage etc.
Slicers and Timelines
Charts for visualization (bar, line, Column, Donut, Area, Funnel)
